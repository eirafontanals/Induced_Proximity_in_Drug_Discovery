# Induced Proximity in Drug Discovery: Molecular Stabilizers, Molecular Glues, and PROTACs

## Overview

This repository contains the complete material developed for the final **DEBM** project on **induced proximity in drug discovery**, with a dual focus on **drug discovery concepts** and **structural bioinformatics analysis**.

In addition to comparing the mechanisms of **molecular stabilizers**, **molecular glues**, and **PROTACs**, the project places strong emphasis on the **structural interpretation of these strategies through PyMOL-based visualization**. The repository therefore combines scientific background material, presentation content, structural figures, mechanistic schematics, and supporting documentation in a single organized workspace.

The objective of the project is to explain how small molecules can do more than inhibit a protein target: they can also stabilize protein conformations, induce productive protein–protein interactions, or trigger targeted protein degradation. These ideas are explored from both a **conceptual** and a **structural bioinformatics** perspective.

---

## Project Scope

The repository is centered on three major induced proximity strategies:

- **Molecular stabilizers**  
  Small molecules that preserve or reinforce a functional protein state.

- **Molecular glues**  
  Monovalent compounds that induce or strengthen protein–protein interactions.

- **PROTACs**  
  Bifunctional molecules that connect a target protein to an E3 ligase in order to promote ubiquitination and degradation.

Beyond the pharmacological comparison, the project also examines how **three-dimensional structural information** helps explain binding mode, molecular recognition, interaction interfaces, and mechanistic differences between these modalities.

---

## Structural Bioinformatics Focus

A central component of this repository is the use of **PyMOL** as a structural bioinformatics tool for the visual interpretation of drug action.

The PyMOL material included in this project was prepared to:

- visualize ligand-binding pockets and molecular interfaces
- compare classical inhibition with induced proximity strategies
- highlight stabilizing interactions in protein complexes
- illustrate glue-mediated ternary recognition events
- depict PROTAC-related target–ligase assemblies
- generate publication-style images suitable for academic presentation

This structural perspective is essential to the project because induced proximity is not only a pharmacological concept, but also a **spatial and molecular recognition problem**. The repository is therefore especially useful for students or researchers interested in the intersection of **chemical biology**, **drug discovery**, and **structural bioinformatics**.

## Folder Description

### `Papers/`
Collection of scientific articles used as the literature foundation of the project. These papers support the theoretical discussion, mechanistic interpretation, and translational relevance of the therapeutic strategies presented.

### `Labels/`
Drug labels and regulatory documentation for compounds discussed in the project. These files provide applied and clinical context for selected examples.

### `Pymol/`
Repository of molecular structure images generated in **PyMOL**.

This folder includes visual material related to:

- inhibitor binding pockets
- transthyretin stabilization by tafamidis
- glue-mediated interactions
- PROTAC ternary complex visualization
- hydrophobic and polar surface interpretation
- close-up interaction views prepared for teaching and presentation purposes

These images form the structural core of the project and reflect its bioinformatics-oriented perspective.

### `RStudio/`
Material generated in **RStudio** to support the visual and mechanistic explanation of induced proximity strategies.

This folder includes:

- **`.Rmd` files** with the source code used to build the figures
- **`.html` files** containing rendered outputs for easy review
- **`.png` files** exported for inclusion in the presentation

The content of this folder is especially useful for comparative schematics and ubiquitination/degradation pathway illustrations.
